Definition List - HTML Terms

HTML
HyperText Markup Language - the standard language for creating web pages.
CSS
Cascading Style Sheets - used to style and layout web pages.
Tag
An HTML element written inside angle brackets, like <h1> or <p>.
Attribute
Extra information added to an HTML tag, like href in <a href="">.
